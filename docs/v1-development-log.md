# V1 Development Log

## Current direction

The game is built around a memory timeline from middle school to university.

## Design rules

- Puzzle logic must be independent from story content.
- Visual references are transformed into original motifs.
- Emotional expression remains commemorative and subtle rather than direct confession.
- Future real memories can be inserted through configuration.

## Current modules

- story timeline
- puzzle registry
- cipher engine
- state management
- UI foundation
