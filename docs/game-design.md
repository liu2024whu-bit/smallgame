# 《1029 · 第20圈年轮》初版设计说明

## 1. 核心定位

这不是一款要求玩家“逃出去”的密室，而是一份可继续生长的生日礼物。树是整个项目的内容框架：

- 树根保存日期与最早的线索
- 枝条承载初三到大三的成长卡
- 花朵把情绪和单词转成可互动机关
- 年轮代表年龄与时间
- 花苞代表尚未发生、以后再补的章节

初版预计首次游玩 15–30 分钟，主线难度为低到中等；每个机关有三层提示，避免因单点卡关中断完整体验。

## 2. 主线节奏

| 阶段 | 玩家动作 | 逻辑类型 | 得到的答案 |
| --- | --- | --- | --- |
| 树根 | 观察日历并转换日期格式 | 环境联想 | 1029 |
| 时间线 | 排序七个学段，再按位置提取 | 排序 + 提取 | LXY |
| 花圃 | 根据花语诗句点击花朵 | 语义映射 + 顺序 | BIRTHDAY |
| 年轮 | 理解“十九圈 + 今天的新一圈” | 数量推理 | 20 |
| 礼物 | 合并四条已知答案 | 信息回收 | 祝福信 |

难度曲线先用简单日期机关教学，再进入时间线的两步推理；花语机关提供情绪变化，年轮是主题性的短谜题，最终礼物只做信息回收，不强行增加新规则。

## 3. 纪念意义

故事文本采用“留白式纪念”而不是替两个人定义关系：

- 明确从初三认识、走到大三和20岁生日
- 不虚构具体事件或浪漫关系
- 最终信中说明“很多细节没有写进初版”，为真实回忆保留空间
- 三种结局全部正向，分别强调未来、纪念与并肩

## 4. 猫狗与忍者彩蛋

猫和狗服务主线逻辑：

- 猫的项圈提供提取位置 `2、5、7`
- 狗的吊牌提供读取方向“从最早到现在”

带土 / 卡卡西元素只作为可选手账彩蛋：用橙色旋涡、迟到的钟、银色雷光与遮住的眼做原创抽象符号。它不影响通关，避免不熟悉作品的玩家被卡住；完成后解锁“并肩”结局。

## 5. 视觉方向

- 外层界面使用浅米白、鼠尾草绿、淡桃色与柔和金色，强调清新手绘和纸张质感
- 场景是月下庭院，但整体亮度高于传统恐怖密室
- 中央树、花、猫狗与物件使用仓库内原创 SVG / CSS 图形
- 动画只用于树冠呼吸、花朵轻摆、星光闪烁和完成状态反馈
- 支持 `prefers-reduced-motion`

## 6. 可扩展架构

`src/content.js` 与玩法代码分离：

```js
STORY            // 文案与人物配置
TIMELINE_CARDS   // 可继续追加年份与真实回忆
FLOWERS           // 花语、图形与字母片段
PUZZLE            // 答案和提取规则
ENDING_COPY       // 结局文本
```

新增章节时建议继续采用数据驱动：每个枝条对应一个章节配置，每个章节注册自己的热点、谜题状态与完成奖励；主树只负责显示章节是否生长。

## 7. 下一版建议

1. 添加真实照片，但通过点击、拼图、遮罩或局部放大参与解谜，不把照片当普通轮播。
2. 将“初三—高中—大学”拆为三条枝条，各自拥有一个短机关。
3. 添加语音或音乐时保留无声音替代文本，保证可访问性。
4. 加入一年一圈的内容更新机制，未来生日只需追加配置和一个章节。
