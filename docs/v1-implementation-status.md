# V1 Implementation Status

## Goal

Build a complete birthday puzzle game with a continuous timeline from junior high school to university.

## Chapter flow

2006 birth clue -> sky map -> breathing pattern -> 104 film archive -> Wuhan/Nanjing postmark -> twentieth ring.

## Design principles

- LoveChoice-like emotional pacing: memories are discovered naturally.
- Rusty Lake-like observation and combination puzzles.
- No direct character collage. External inspirations are transformed into symbols and mechanisms.
- Romantic but not confession-focused: the core is remembering growth and wishing a happy twentieth birthday.

## Current architecture

story: narrative data
puzzles: independent puzzle definitions
engine: verification and unlocking
state: save and progress
ui: rendering layer
