# 写给第20圈的无地址信

一款为即将20岁的朋友制作的原创网页解谜生日礼物。故事沿着“出生—初三相识—高中—大一—大二—大三—20岁”展开，以照片、角色立绘、原创画稿、校园影像与真实花朵组成连续的记忆册。

情感表达以纪念与祝福为核心：带一点浪漫主义色彩，但不替两个人定义关系，也不把结尾写成强行表白。

## 当前正式主线

1. 猫狗照片与吊牌 → `2006`
2. 光遇造型、樱花与玫瑰照片排序 → `2020`
3. 魈与流浪者立绘中的四枚风窗 → `EAST`
4. 水柱、风柱、蛇柱与紫藤照片排序 → `BIRTHDAY`
5. 收礼人原创画稿上的104号观察机关 → `LXY`
6. 武汉大学与南京大学校园照片、植物邮戳 → `1029`
7. 真实年轮照片与三张年份卡 → `20`
8. 花束与礼物盒三重锁 → `06102920 / LXY / BIRTHDAY`

104号底片章节直接使用收礼人自己的带土、卡卡西、钥匙扣与猫咪作品，并在画面上设置红色焦点、负片切换和猫咪观察热点。其余章节不再用简单几何符号代替主体内容。

## 图像体系

所有图像入口集中在 [`src/assets.js`](src/assets.js)：真实花朵、猫狗、校园照片、角色立绘、光遇造型图，以及 `assets/art/` 中收礼人的原创作品。替换图片时只需修改素材清单，不需要进入章节逻辑。来源及替换注意事项见 [`docs/asset-sources.md`](docs/asset-sources.md)。角色素材仅用于这份非商业生日礼物的展示，版权归原权利人所有。

## 架构

```text
index.html                 静态入口与启动错误兜底
styles/                    基础、布局、照片场景、热点和响应式样式
src/assets.js              全部图像素材及替换入口
src/content.js             故事、章节、提示、结局和未来插槽
src/puzzles.js             纯谜题规则与答案验证
src/store.js               自动存档与状态更新
src/runtime.js             通用运行时和章节工具
src/render.js              应用外壳渲染
src/render-chapters.js     各章节图像场景与谜题视图
src/dialogs.js             序章、提示、旧锁和隐藏手账弹窗
src/app.js                 交互编排与统一事件委派
tests/                     逻辑、模块、图像清单与渲染测试
assets/art/                经裁切压缩的原创作品素材
```

故事、谜题、状态、素材和界面彼此分离。补真实经历时编辑 `src/content.js`；替换图像编辑 `src/assets.js`；修改密码方向编辑 `src/puzzles.js`，不把答案散落到点击事件中。
