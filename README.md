# 写给第20圈的无地址信

一款为即将20岁的朋友制作的原创网页解谜生日礼物。游戏沿着“出生—初三相识—高中—大一—大二—大三—20岁”展开，使用猫狗吊牌、星章、四阵风、呼吸谱、104号底片、武汉—南京双城邮戳和年轮构成连续谜题链。

情感表达以纪念与祝福为核心：带一点浪漫主义色彩，但不替两个人定义关系，也不把结尾写成强行表白。

## 当前正式主线

1. 猫狗吊牌 → `2006`
2. 初三星图 → `2020`
3. 高中四阵风 → `EAST`
4. 大一呼吸谱 → `BIRTHDAY`
5. 大二104号底片 → `LXY`
6. 武汉大学 × 南京大学双城邮戳 → `1029`
7. 三段年轮 → `20`
8. 最终锁 → `06102920 / LXY / BIRTHDAY`

大二章节使用收礼人自己的带土、卡卡西与猫咪作品。其他作品元素采用抽象符号、纹样和机关表达，避免不同画风的角色立绘直接堆叠。

## 运行

```bash
npm run serve
```

打开 `http://localhost:4173`。

测试：

```bash
npm run check
npm test
```

## 架构

```text
index.html                 静态入口
styles/                    统一设计变量、章节视觉和响应式样式
src/content.js             故事、章节、提示、结局和未来插槽
src/puzzles.js             纯谜题规则与答案验证
src/store.js               自动存档与状态更新
src/runtime.js             通用运行时和章节工具
src/render.js              应用外壳渲染
src/render-chapters.js     各章节界面渲染
src/dialogs.js             序章、提示、旧锁和隐藏手账弹窗
src/app.js                 交互编排与统一事件委派
tests/                     纯逻辑测试
assets/art/                经压缩的原创作品网页素材
```

故事、谜题、状态和界面分离。后续补真实经历时优先编辑 `src/content.js`；修改密码方向时优先编辑 `src/puzzles.js`，不需要把答案散落到点击事件里。
