# 第20圈闭合以前

当前开发线：v1.0

这是一份为即将20岁的朋友制作的原创网页解谜生日礼物。目标不是制作一封直白的表白信，而是通过一场关于时间、作品、城市与记忆的解谜，让玩家重新走过从初三到大三的时间线。

## 版本策略

- `main`：稳定试玩版本，用于 GitHub Pages。
- `develop-v1`：下一代开发版本，完成测试后合并。
- 历史版本保留，不直接覆盖旧体验。

## v1.0主线规划

```
初三 → 光遇、小王子、玫瑰、星空
高中 → 风、羽毛、少年旅途
大一 → 呼吸纹理、花与四季
大二 → 104号底片、原创作品、猫与双人手账
大三 → 武汉与南京两座城市
20岁 → 第20圈年轮
```

## 核心设计原则

- 解谜参考点击式探索游戏：观察 → 联想 → 组合 → 回收。
- 情感参考温柔叙事游戏：克制、留白、不强行定义关系。
- 所有元素必须服务故事，不作为简单彩蛋堆叠。

## 技术方向

新版采用模块化架构：

```
story       故事与时间线
puzzles     谜题配置
engine      密码算法与状态逻辑
state       存档管理
ui          界面组件
```

避免将剧情、答案、界面事件混合，保证后续增加真实故事、照片、语音和新章节时可以继续维护。

## 后续扩展

等待补充真实经历后，将把对应章节替换为更个人化内容：

- 共同经历
- 特殊日期
- 照片机关
- 手写信
- 音频或隐藏彩蛋
