# Changelog

## 1.1.0

- 全部主线章节改为真实照片、角色立绘或收礼人原创画稿构成的场景。
- 花束、章节花朵、猫狗、校园、年轮与礼物视觉改为图像驱动。
- 高中章节加入魈与流浪者立绘，大一章节加入水柱、风柱与蛇柱角色图。
- 104号底片改为在作品画面上直接寻找热点、负片和第四组猫咪动作。
- 新增统一素材清单 `src/assets.js`，后续替换图像无需修改谜题代码。
- 新增图像资源与章节渲染测试。
